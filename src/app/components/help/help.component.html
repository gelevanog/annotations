Справка по работе с аннотациями

Что такое аннотации?

Аннотации позволяют выделять фрагменты текста в статьях и добавлять к ним примечания. Каждая аннотация имеет цвет подчеркивания и текстовое примечание, которое отображается при наведении курсора на выделенный текст.

Создание статьи

  1. На главной странице нажмите кнопку "+ Новая статья"
  2. Статья создастся автоматически с заголовком "Без названия"
  3. Введите заголовок статьи в верхнем поле
  4. Начните вводить текст в области содержимого
  5. Все изменения сохраняются автоматически

Создание аннотации

  1. Выделите нужный фрагмент текста мышью
  2. Появится форма создания аннотации в правом нижнем углу
  3. Введите текст примечания в поле "Примечание"
  4. Выберите цвет подчеркивания из предложенной палитры
  5. Нажмите кнопку "Добавить аннотацию"
  6. Выделенный текст будет подчеркнут выбранным цветом

Примечание: Кнопка "Добавить аннотацию" будет неактивна, если поле примечания пустое.

Просмотр аннотаций

  • В тексте: Наведите курсор на подчеркнутый текст, чтобы увидеть всплывающую подсказку с примечанием
  • В списке: Все аннотации статьи отображаются в разделе "Аннотации" под текстом статьи
  • В списке показывается выделенный текст, примечание, цвет и дата создания

Пересекающиеся аннотации

Вы можете создавать несколько аннотаций на одном и том же фрагменте текста. В этом случае:

  • Подчеркивание будет отображаться градиентом из всех цветов пересекающихся аннотаций
  • При наведении курсора отобразятся все примечания через разделитель " | "
  • Каждая аннотация сохраняется независимо и может быть отредактирована или удалена отдельно

Редактирование аннотации

  1. В списке аннотаций найдите нужную аннотацию
  2. Нажмите кнопку ✎ (редактировать)
  3. Измените текст примечания и/или выберите новый цвет
  4. Нажмите "Сохранить" для применения изменений
  5. Или нажмите "Отмена" для отмены редактирования

Важно: После изменения цвета аннотации, подчеркивание в тексте автоматически обновится.

Удаление аннотации

  1. В списке аннотаций найдите нужную аннотацию
  2. Нажмите кнопку × (удалить)
  3. Подтвердите удаление в диалоговом окне
  4. Аннотация будет удалена, а подчеркивание исчезнет из текста

Управление статьями

  • Просмотр списка: На главной странице отображаются все созданные статьи
  • Открытие статьи: Кликните на карточку статьи для открытия
  • Удаление статьи: Нажмите кнопку × на карточке статьи и подтвердите удаление
  • Редактирование: Все изменения в заголовке и тексте сохраняются автоматически

Хранение данных

Все статьи и аннотации сохраняются в локальном хранилище браузера (localStorage). Это означает:

  • Данные сохраняются только на вашем устройстве
  • Данные доступны при повторном открытии приложения
  • Данные не синхронизируются между разными браузерами или устройствами
  • При очистке данных браузера все статьи и аннотации будут удалены

Технические особенности

  • Приложение работает полностью в браузере без серверной части
  • Используется Range API для точного определения позиций текста
  • Поддерживается автосохранение при вводе текста
  • Все аннотации восстанавливаются при повторном открытии статьи
